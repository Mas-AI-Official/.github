## What we build

> **Daena is a governed AI control plane and multi-agent "AI company OS": it routes work across 9 LLM runtimes while enforcing a 10-stage governance pipeline, auditable memory, and human-approval gates before any agent action executes.**

- **Daena** — governance-first AI agent orchestration. 10 departments × 6 capabilities = 60 governed agent slots, a 10-stage pipeline (SecurityGate → AuditLog), a 5-tier Neural-Backed Memory Fabric, three reasoning modes, and multi-LLM routing across Claude, GPT, Gemini, Grok, Ollama, Groq, OpenRouter, Together, and Perplexity — hot-swappable without restart.
- **Klyntar** — Daena's security mode. 25+ exploit signatures, behavior-based scanner detection, and a zero-false-positive gate.
- **KYA Mission Control** — trust infrastructure for autonomous AI agents: verifies who owns an agent and what it is allowed to do, with Ed25519-signed, offline-verifiable receipts. → [kya.mas-ai.co](https://kya.mas-ai.co)
